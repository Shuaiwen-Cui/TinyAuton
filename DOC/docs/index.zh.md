# TINYAUTON

![封面](cover.jpg)

## 关于本项目

这个项目致力于开发一个运行在 MCU 设备上的小型智能体相关的计算库，以服务于多智能体系统，涵盖数学运算、数字信号处理和 TinyML。

!!! info "名字的由来"
    "TinyAuton" 是 "Tiny" 和 "Auton" 的组合。"Tiny" 意味着智能体被设计为运行在 MCU 设备上，而 "Auton" 是 "Autonomous Agent" 的缩写。

## 目标硬件

- MCU 设备（目前以 ESP32 为主要目标）

## 覆盖范围

- 平台适配与各类各类工具（时间、通讯等）
- 基本数学运算
- 数字信号处理
- TinyML / 边缘人工智能
