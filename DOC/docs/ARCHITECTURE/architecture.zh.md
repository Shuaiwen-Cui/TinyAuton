# 架构

## 分层架构

```txt
+------------------------------+
| AI                           | <-- 基于低级函数的边缘设备 AI/ML 函数
+------------------------------+
| DSP                          | <-- 数字信号处理函数
+------------------------------+
| Math Operations              | <-- 各种应用的常用数学函数
+------------------------------+
| Adaptation Layer             | <-- 用平台优化/特定函数替换标准 C 中的函数
+------------------------------+
```

## 代码组织

```txt

+------------------------------+
| 应用层                        |
+------------------------------+
| 中间件                        |
|   - TinyAdapter              | <-- 用于屏蔽硬件差异并提供统一 API
|   - TinyMath                 | <-- 常用数学函数
|   - TinyDSP                  | <-- DSP 函数
|   - TinyAI                   | <-- AI 函数
|   - TinyEvaluator            | <-- 评估性能
+------------------------------+
| 驱动层                        |
+------------------------------+
| 硬件层                        |
+------------------------------+

```
